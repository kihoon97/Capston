에러코드란(Erorr Code, Response Code)
서버에서 브라우저로 발신되는 http상태코드 중 하나로, 리스폰스 코드라고도 한다.

웹사이트에 에러가 있을 경우, 서버에서 브라우저로 보내는 번호를 뜻한다.
대중적으로 접할 수 있는 erorr 페이지
  • 401 : Unauthorized (접근권한이 없습니다)
  • 404 : Not Found (페이지를 찾을 수 없습니다)
  • 500 : Internal Server Erorr (서버 문제로 인한 사용 제한)

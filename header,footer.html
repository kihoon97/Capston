header, footer

HTML의 header와 footer 요소는 웹 페이지의 구조를 나타내는데 사용됩니다. 각각 페이지의 맨 위와 맨 아래에 위치하며, 일반적으로 사이트의 로고, 제목, 메뉴 등을 담고 있는 헤더 영역과 페이지의 하단에 있는 저작권 정보, 연락처 등을 담고 있는 푸터 영역을 표현하는 데 사용됩니다.

header의 요소

- header 요소는 웹 페이지나 섹션의 헤더를 정의합니다.

- 헤더는 보통 웹 페이지 상단에 위치하며, 사이트 로고, 제목, 메뉴 등을 포함할 수 있습니다.

- 하나의 페이지에 여러 header 요소가 있을 수 있습니다. 예를 들어, 섹션마다 다른 헤더를 가질 수 있습니다.

footer의 요소

- footer 요소는 웹 페이지나 섹션의 푸터를 정의합니다.

- 푸터는 일반적으로 웹 페이지의 맨 아래에 위치하며, 저작권 정보, 연락처, 사이트 링크 등을 포함할 수 있습니다.

- 하나의 페이지에 여러 footer 요소가 있을 수 있습니다. 예를 들어, 섹션마다 다른 푸터를 가질 수 있습니다.

footer의 요소

- footer 요소는 웹 페이지나 섹션의 푸터를 정의합니다.

- 푸터는 일반적으로 웹 페이지의 맨 아래에 위치하며, 저작권 정보, 연락처, 사이트 링크 등을 포함할 수 있습니다.

- 하나의 페이지에 여러 footer 요소가 있을 수 있습니다. 예를 들어, 섹션마다 다른 푸터를 가질 수 있습니다.

ex) div class="card-header"
div class="card-body"
ex) 카드 헤더를 선언하지 않았을 때
장점
헤더와 푸터 요소를 사용하면 웹 페이지의 구조를 더 명확하게 표현할 수 있으며, 스타일링이나 스크립팅에 유용한 요소들을 묶어서 관리할 수 있습니다. 이를 통해 유지보수가 쉽고 읽기 쉬운 코드를 작성할 수 있습니다.
